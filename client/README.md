# Web photo client
## Requirements
- Node.js version v20.10.0 or higher
- npm version 10.2.3 or higher
### Install dependencies
```bash
yarn
```
### Create .env file
Please create `.env` file from `.env.example` file and change all the environment variables.
```
touch .env
cp .env.example .env
```
### Run client
```bash
yarn start
```
